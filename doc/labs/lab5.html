Examine and Test the Tree Builder

  • (25 min) Try many examples like 
          >> xcom -treeTrace -noExecute x:=1
      >> xcom -treeDump -noExecute x:=1
      >> xcom -treeDump -noExecute -noAST x:=1
      >> xcom -treeDump -noExecute 'if x<0? y:=1; :: x>0? y:=2; fi'
    For each generated output, draw the tree.

     

  • (05 min) Add a (better) unit test to testTree.m 
          >> edit mxcom/m/tests/testTree.m

Examine and Test the Symbol Table

  • (10 min) Try many examples like 
          >> xcom -symTrace -noExecute x:=1
      >> xcom -symDump -noExecute xyz:=xyz+1.1
      >> xcom -symDump -noExecute b1:=b2|true
      >> xcom -symDump -noExecute r:=i2r(17)
      >> xcom -symDump -noExecute 'if x<0? y:=1 :: x>0? y:=2 fi'
  • (20 min) Add a unit test to testSymbols.m 
          >> edit mxcom/m/tests/testSymbols.m
