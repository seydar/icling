gem.m

  • (10 min) Look at gem.m 
          >> edit gem.c
  • (20 min) Try examples like 
          >> G=gem()
      >> G.nowhite
      >> scan=@(txt)G.run(txt, G.nowhite);
      >> scan('x Y z')
      >> parse = G.run('x+x+x', scan(G.sum))
      >> unsum = G.run(scan(G.sum), scan(G.invert))
      >> G.run(unsum, scan(G.pretty))
      >> raw = G.run(parse, unsum)
