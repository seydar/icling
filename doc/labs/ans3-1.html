The structure of the tests suggests adding a new case 

  if allt || any(tn == 3)              % my test
  ...stuff
  end

The point of the if-end is that you can easily run all or some subset of the tests. You will be interested in running test 3.

Then you need to decide what kind of test you want. Among the virtually unlimited possibilities are

  • all pairs of kinds of lexemes (A B) (A 1) (A+) (+ +) (++)
  • limits 44444444444444444444444444444444444444444444444
  • bad stuff #@[]{}...
