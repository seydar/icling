A Lexical CFG

Not including the definitions of letter, digit, notEOL and EOL, here is a grammar for the X lexicon. The advantage of the description is precise description of where white space is required, how d3 will be interpreted, etc.

Typically the precision here is already present in the lexing code that is produced by hand, so this description serves little purpose for xcom. It can be important for a language with a much more complicated and error prone lexical structure.

There similar descriptions used as input to lexer generators (such as lex).
