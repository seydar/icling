Future Plans

This compiler course is a work in progress. It is usable now, but will undergo improvements for some time. It is my hope that users will communicate their desires and advice to me. Here is a list of the known major work items.

  1. Port to Intel A64.
  2. Much better unit tests.
  3. Upgrade powerpoint LALR(1) presentation to LR(1)

Earlier Versions

There is an earlier version of xcom written in C++ for linux. It is harder to learn than the MATLAB version but is more conventional as a system program.

There is an even earlier version of xcom written in C for linux. It is called hyper on account of its speedy compile time. It includes an emacs-like editor from which all compile&execute commands are issued.

Both versions are available as zipped email attachments upon request.
